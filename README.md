# Портфолио разработчика

Персональное портфолио (визитная карточка) разработчика, созданное на Next.js.

## Стек технологий

- **Next.js** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **React Hooks** - интерактивность

## Структура проекта

portfolio/
- app/
  - layout.tsx - Основной макет с SEO
  - page.tsx - Главная страница
- components/
  - Header.tsx - Шапка с навигацией
  - Hero.tsx - Приветственный блок
  - About.tsx - Раздел Обо мне
  - Projects.tsx - Раздел Проекты
  - Contacts.tsx - Раздел Контакты
  - Footer.tsx - Подвал
- styles/
  - globals.css - Глобальные стили
- public/ - Статические ресурсы
- package.json

## Установка и запуск

### Локальная разработка

1. Установка зависимостей: npm install
2. Запуск сервера разработки: npm run dev
3. Откройте http://localhost:3000 в браузере

### Сборка для продакшена

1. Создание продакшен сборки: npm run build
2. Запуск продакшен сервера: npm run start

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Запушьте код в репозиторий
3. Включите GitHub Pages в настройках репозитория (Settings > Pages)
4. Выберите ветку gh-pages или настройте GitHub Actions для автодеплоя

## Настройка

### Изменение контактной информации

Откройте components/Contacts.tsx и измените массив socialLinks.

### Добавление проектов

Откройте components/Projects.tsx и измените массив projects.

### Изменение информации о себе

Откройте components/About.tsx и измените массивы skills и experiences.

## Производительность

Проект оптимизирован для быстрой загрузки:
- Статическая генерация страниц (SSG)
- Оптимизация изображений
- Минимизация CSS через Tailwind
- Код-сплиттинг Next.js
